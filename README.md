# Local Food Lovers Network

---

This is the backend API for the Local Food Lovers Network, a MERN-stack web application where users can discover foods, write reviews, save favorites, and explore top-rated dishes from local restaurants.

The server is built using Node.js, Express.js, and MongoDB, with secure environment variable handling, validated CRUD operations, and optimized endpoints for production-ready performance.
## Features

- Users can register and fetch their own data

- Add new food reviews with full details

- Search reviews by food name

- Update and delete personal reviews

- Add and remove favorites


## Tech Stack

- Node.js

- Express.js

- MongoDB (MongoDB Atlas)
